# My First Project

## Что это за проект

Личный мини-сайт, собранный из отдельных HTML-страниц — без фреймворков и сборщиков.
Каждая страница самодостаточна: весь CSS и JS вшиты прямо в файл.
Проект развивается итерационно: новые инструменты и страницы добавляются по мере необходимости.

## Для кого

Для личного использования и демонстрации: визитка и лендинг — для потенциальных клиентов, трекер привычек — для себя.

## Страницы, которые уже есть

| Файл | Название | Описание |
|------|----------|----------|
| `landing.html` | BeautyBot | Лендинг ИИ-администратора для салонов красоты. Записывает клиентов 24/7, ориентирован на Орехово-Зуево. Есть CTA, описание функций, тариф, FAQ. |
| `vizitka.html` | Визитка | Личная/деловая визитка. |
| `habits.html` | Трекер привычек | Недельный трекер 5 привычек (вода, зарядка, чтение, сон до 23:00, без сахара). Данные хранятся в `localStorage`, есть прогресс-бары по дням и общий за неделю. |

## Страницы, которые планируются

- Портфолио / кейсы
- Страница с контактами
- Дашборд с несколькими трекерами

## Стиль дизайна

Два визуальных направления, используемых в проекте:

**BeautyBot (landing.html)**
- Тёмный фон (`#070714 → #130a1f`)
- Акценты: розовый `#f472b6` и фиолетовый `#a855f7`
- Шрифт: Manrope (Google Fonts)
- Стиль: glassmorphism, мягкие градиенты, карточки с backdrop-blur

**Трекер привычек (habits.html)**
- Тёмный фон (`#0d0d1a`)
- Акценты: cyan `#00f5ff` и фиолетовый `#bf00ff`
- Шрифт: Segoe UI / system-ui
- Стиль: киберпанк / неоновое свечение, ripple-эффекты, анимированные progress-bars
