# Правила работы с проектом

## Структура файлов

- Каждая страница — один самодостаточный HTML-файл: весь CSS и JS внутри, никаких внешних файлов.
- Никаких фреймворков и библиотек — только ванильный HTML, CSS, JavaScript.

## Дизайн

- Фон — тёмный, текст — светлый.
- Акцентный цвет — оранжевый.
- Адаптивная вёрстка обязательна: страница должна хорошо выглядеть на мобильных устройствах.

## Стиль текста

- Простой и дружеский язык, без канцелярита.
- Коротко и по делу.

## Перед созданием нового файла

- Сначала описать, что планируется сделать: какой файл, что будет внутри, какова структура.
- Дождаться подтверждения, и только потом писать код.
